Flipkart Scraper
===============
This is a simple flipkart scraper developed in python.It scrapes data and dumps it into result.json file.

Prerequisites:
==============
1) BeautifulSoup4 
2) requests
3) json

Installation:
============
git clone 
cd flipkart-scraper

Usage:
=====
python3 scrape-products.py
Enter a query: dell laptops
