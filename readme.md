Flipkart Scraper
===============
This is a simple flipkart scraper developed in python.It scrapes data and dumps it into result.json file.

Prerequisites:
==============
1) BeautifulSoup4 
2) requests
3) json

Installation:
============
1) git clone 
2) cd flipkart-scraper

Usage:
=====
1) python3 scrape-products.py
2) Enter a query: dell laptops
